Is This Song Clean?

Enter a song title and artist and find out its cleanliness.




Why should I use this site if Spotify/iTunes/Google Music tells me which songs are explicit?

While it is true that these services do provide a single flag for the explicitness of songs, they suffer from three separate problems:

  • Explicit tags are done by humans and can often be incorrect and miss curse words in the music.
  • These tags work as an on/off switch, rather than the multiple layers of cleanliness that Is This Song Clean provides for various social settings.
  • These tags only rely on strictly explicit language, while Is This Song Clean can take user input to identify implied offensive material.
